# Python Basics

Collection of Python exercises developed during my fundamentals training, covering core concepts such as loops, lists, dictionaries, and functions, with an introduction to pandas for basic data manipulation.

### Repository structure
- python_basics.ipynb → general introduction to Python syntax and basic concepts
- practicas_for_def_dict_list.ipynb → exercises on loops, functions, lists, and dictionaries
- pandas_basics.ipynb → introduction to pandas and basic operations
- practicas_pandas.ipynb → hands-on exercises applying pandas for data manipulation

### Topics covered
- Variables and data types
- Control flow (if/else, loops)
- Lists and dictionaries
- Functions
- Data manipulation with pandas

### Purpose
Build a solid foundation in Python programming before moving into more advanced data analysis tasks using real datasets and libraries such as pandas.

### Tools
Python 3
Jupyter Notebook
pandas (basic usage)
